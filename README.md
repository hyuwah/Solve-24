## Requirements
Python 2.7

PIP (for easy package installation)

PyQt

PyInstaller

## Build
`$ git clone https://github.com/hyuwah/Solve-24.git`

1. Navigate to project folder

2. open shell and type `pyinstaller --onefile --window solve24.py`
